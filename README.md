# ClashFox - Native Proxy

## Overview

ClashFox is a modern Electron-based GUI that provides a user-friendly interface for managing the Mihomo (Clash Meta) kernel. It offers a comprehensive toolkit for configuration, monitoring, and control, with native macOS integration and advanced features.

## Features

- **Modern GUI**: Clean and intuitive Electron-based interface
- **Kernel Management**: Full control over Mihomo kernel lifecycle
- **Configuration Editor**: Powerful config editing with syntax highlighting
- **Real-time Logs**: Live log monitoring and filtering
- **MacOS Integration**: Native LaunchDaemon helper for background service
- **Foxboard Dashboard**: Advanced traffic monitoring and analytics
- **Tracker Support**: Connection tracking and statistics

## Requirements

- Node.js 20+
- npm 10+
- macOS (recommended for full toolkit/helper capability)
